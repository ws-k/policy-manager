# DWFM 서비스 정책 명세서

> 화면별 유저 인터랙션 정책. 비개발자도 읽을 수 있도록 작성.

## 검색 화면 정책


### 유저가 검색 화면에 직접 진입하면 키보드가 자동으로 열립니다

- 검색창에 자동으로 커서가 위치하고 키보드가 열립니다
- 홈 화면 검색 바 탭으로 진입한 경우에만 해당됩니다


### 유저는 검색창에 매장명·카테고리·호수를 입력해 검색할 수 있습니다

- 매장명, 카테고리, 세부 카테고리, 호수 코드를 기준으로 검색합니다
- 키보드의 검색 버튼을 누르면 검색이 실행됩니다


### 유저가 카테고리를 탭해 진입하면 해당 카테고리로 즉시 검색이 실행됩니다

- 홈 화면 카테고리 탭 → 검색 화면 진입 시 해당 카테고리 검색이 자동 실행됩니다
- 이 경우 키보드는 자동으로 열리지 않습니다


### 유저가 ✕ 버튼을 탭하면 검색 내용이 모두 초기화됩니다

- 입력된 검색어와 검색 결과가 초기화됩니다
- 선택된 동 필터도 초기화됩니다
- 최근 검색어·인기 검색어 화면으로 돌아갑니다


### 유저가 검색어를 입력하면 자동완성 제안이 표시됩니다

- 입력 중인 검색어와 관련된 제안이 최대 6개 표시됩니다
- 인기 검색어 우선, 이후 매장명·카테고리·세부 카테고리 순으로 표시됩니다


### 유저가 자동완성 항목을 탭하면 즉시 검색이 실행됩니다

- 해당 검색어로 바로 검색이 실행됩니다
- 키보드가 닫힙니다


### 유저는 최근 검색어를 탭해 빠르게 검색할 수 있습니다

- 최근 검색어를 탭하면 바로 검색이 실행됩니다
- 최근 검색어는 현재 고정값으로 표시됩니다 (실제 기기 저장 연동은 추후 구현 예정)


### 유저는 인기 검색어를 탭해 빠르게 검색할 수 있습니다

- 인기 검색어를 탭하면 바로 검색이 실행됩니다
- 인기 검색어 목록: 면직물, 폴리에스터, 레이스, 니트, 실크, 지퍼, 단추, 안감, 자수, 데님


### 유저가 동 필터 칩을 탭하면 해당 동의 매장만 필터링됩니다

- 전체, A동, B동, C동, 신관 중 하나를 선택할 수 있습니다
- 선택 즉시 결과가 필터링됩니다
- 동 필터는 하나만 선택 가능합니다 (중복 선택 불가)


### 유저가 매장 카드를 탭하면 해당 매장 상세 화면으로 이동합니다

- 카드에는 매장 코드, 이름, 동·카테고리가 표시됩니다
- 2열 카드 그리드로 표시됩니다


### 검색 결과가 없을 때 유저에게 안내 문구가 표시됩니다

- 화면 중앙에 "검색 결과가 없습니다" 문구가 표시됩니다


### 유저가 "지도에서 보기" 버튼을 탭하면 지도 화면으로 이동합니다

- 검색 결과가 1개 이상일 때만 버튼이 표시됩니다
- 현재 검색어는 지도로 전달되지 않습니다

## 내위치 확인 정책


### 유저가 드럼 피커에서 동을 선택하면 해당 동의 층 목록이 표시됩니다

- A동, B동, C동, 신관(N동) 중 하나를 스크롤해서 선택합니다
- 동을 변경하면 층 목록이 자동으로 바뀝니다
- 이전에 선택했던 층이 새 동에 없으면 첫 번째 층으로 자동 초기화됩니다
- 동 변경 시 호수 입력란도 초기화됩니다


### 유저가 드럼 피커에서 층을 선택합니다

- 선택된 동의 사용 가능한 층 목록 중 하나를 스크롤해서 선택합니다
- 층 변경 시 호수 입력란이 초기화됩니다


### 동별 사용 가능한 층 목록입니다

- A동: 지하1층, 1층~7층
- B동: 지하1층, 1층~6층
- C동: 지하1층, 1층~6층
- 신관(N동): 지하1층, 1층~9층


*(정보 테이블 — Tiptap 원본 보존)*


### 유저가 호수를 입력합니다

- 숫자 키보드로 최대 4자리 호수를 입력합니다
- 호수 입력 변경 시 이전 오류 상태가 초기화됩니다


### 유저가 확인 버튼을 탭하면 입력한 위치를 지도에서 찾습니다

- 선택된 동·층·입력된 호수에 해당하는 매장을 검색합니다


### 매장을 찾은 경우 지도에 핀이 표시됩니다

- 해당 층의 지도로 자동 전환됩니다
- 지도 화면이 핀 위치로 자동 이동합니다
- 바텀시트가 닫힙니다
- 층당 핀은 1개만 설정 가능하며, 새로 설정하면 기존 핀이 교체됩니다


### 호수를 찾을 수 없는 경우 오류 안내가 표시됩니다

- 입력 필드 아래에 오류 문구가 표시됩니다
- 입력 필드 테두리가 빨간색으로 바뀝니다


### 유저가 "핀 제거" 버튼을 탭하면 지도에서 핀이 삭제됩니다

- 핀이 설정된 상태에서만 "핀 제거" 버튼이 표시됩니다
- 탭하면 지도에서 핀이 사라지고 버튼이 원래 상태로 돌아옵니다


### 유저가 배경을 탭하면 내 위치 설정 창이 닫힙니다

- 창 바깥 영역(배경 딤)을 탭하면 창이 닫힙니다


### 유저는 지도에서 설정된 핀을 시각적으로 확인할 수 있습니다

- 핀이 설정된 층에서 해당 위치에 핀 마커가 표시됩니다

- 헤더 좌측 뒤로가기 버튼을 탭합니다


### 유저는 업무공간 목록을 카드 그리드로 볼 수 있습니다

- 셰어라운지, 무신사스튜디오, 방송스튜디오, 사진스튜디오, 미팅룸 4인, 미팅룸 6인, 디자이너라운지 (총 7개)
- 모든 공간은 4층에 위치합니다
- 각 카드에 이모지, 공간명, 설명, 위치가 표시됩니다


*(정보 테이블 — Tiptap 원본 보존)*


### 유저가 공간 카드를 탭하면 해당 시설 상세 화면으로 이동합니다

- 카드를 탭하면 해당 시설의 예약 또는 상세 화면으로 이동합니다

## 지도 화면 검색 정책


### 유저가 검색 바를 탭하면 전체 화면 검색 오버레이가 열립니다

- 검색창이 자동으로 포커스되고 키보드가 열립니다


### 유저가 "내 위치" 버튼을 탭하면 내 위치 설정 창이 열립니다

- 동·층·호수를 입력해 지도에 위치 핀을 표시할 수 있는 창이 열립니다


### 유저가 뒤로가기 버튼을 탭하면 이전 화면으로 이동합니다

- 지도 화면 이전 화면으로 이동합니다


### 유저가 지도 배경을 탭하면 검색 바 등 상단 UI가 숨겨집니다

- 플로팅 헤더, 건물 칩이 숨겨집니다
- 다시 탭하면 다시 나타납니다
- UI가 숨겨진 상태에서는 버튼이 동작하지 않습니다


### 유저가 검색창에 검색어를 입력합니다

- 매장명, 카테고리, 세부 카테고리, 호수 코드를 기준으로 검색합니다
- 검색 결과는 최대 30개까지 표시됩니다
- 결과가 없으면 "검색 결과가 없습니다" 문구가 표시됩니다


### 유저가 검색 결과에서 개별 매장을 탭하면 해당 매장만 지도에서 강조 표시됩니다

- 검색 오버레이가 닫힙니다
- 지도에서 해당 매장만 강조되고 나머지는 흐리게 표시됩니다


### 유저가 "지도에서 보기" 버튼을 탭하면 전체 검색 결과가 지도에서 강조 표시됩니다

- 검색 오버레이가 닫힙니다
- 지도에서 검색된 모든 매장이 강조됩니다


### 유저가 키보드 검색 버튼을 누르면 전체 검색 결과가 지도에서 강조 표시됩니다

- 검색 오버레이가 닫힙니다


### 유저가 검색창의 × 버튼을 탭하면 검색어가 초기화됩니다

- 검색어가 지워지고 결과 목록이 사라집니다


### 유저가 검색 오버레이의 뒤로가기를 탭하면 오버레이가 닫힙니다

- 검색어도 함께 초기화됩니다


### 검색이 적용된 상태에서 지도 상단에 배지가 표시됩니다

- 검색된 매장 수가 배지에 표시됩니다


### 유저가 배지의 × 버튼을 탭하면 강조 표시가 해제됩니다

- 모든 매장이 일반 상태로 돌아옵니다


### 유저가 필터 아이콘을 탭하면 필터 설정 창이 열립니다

- 카테고리, 동, 층을 조합해 필터링할 수 있습니다


### 유저가 대분류 카테고리를 탭하면 해당 세부 항목 목록이 표시됩니다

- 직물, 니트, 레이스, 피혁, 실, 단추, 지퍼, 부자재 중 하나를 선택합니다
- 대분류는 최소 1개 이상 선택되어야 합니다
- 대분류 변경 시 세부 카테고리 선택이 초기화됩니다


### 유저가 세부 카테고리를 탭해 선택하거나 해제합니다

- 여러 개를 동시에 선택할 수 있습니다
- 이미 선택된 항목을 탭하면 선택이 해제됩니다


### 유저가 동을 탭해 선택합니다

- 특정 동을 선택하면 해당 동의 층 목록이 표시됩니다
- "전체"를 탭하면 동 선택이 초기화됩니다


### 유저가 층을 탭해 선택하거나 해제합니다

- 여러 개를 동시에 선택할 수 있습니다
- "전체"를 탭하면 층 선택이 초기화됩니다
- 동을 선택하지 않은 경우 B1~9층 전체 목록이 표시됩니다


### 유저가 "지도에서 보기" 버튼을 탭하면 필터가 적용됩니다

- 필터 조건에 맞는 매장이 지도에서 강조 표시됩니다
- 필터 창이 닫힙니다


### 유저가 "초기화" 버튼을 탭하면 모든 필터가 초기화됩니다

- 대분류는 첫 번째 항목(직물)으로 초기화됩니다
- 세부 카테고리, 동, 층 선택이 모두 초기화됩니다

## 지도 화면 탐색 정책


### 유저가 두 손가락으로 지도를 확대·축소할 수 있습니다

- 두 손가락을 모으면 지도가 축소됩니다
- 두 손가락을 펼치면 지도가 확대됩니다
- 최대·최소 줌 범위가 제한되어 있습니다


### 유저가 한 손가락으로 드래그해 지도를 이동할 수 있습니다

- 드래그한 방향으로 지도가 이동합니다


### 유저가 지도 배경을 탭하면 상단 UI가 숨겨집니다

- 플로팅 헤더, 층 선택기, 건물 칩이 숨겨집니다
- 다시 탭하면 UI가 다시 나타납니다


### 유저가 건물 칩에서 "전체"를 탭하면 모든 동이 동일하게 표시됩니다

- 특정 동 강조가 해제되고 모든 매장이 동일한 색상으로 표시됩니다


### 유저가 특정 동을 탭하면 해당 동의 매장이 강조됩니다

- 해당 동의 매장이 진하게 표시됩니다
- 다른 동의 매장은 흐리게 표시됩니다
- 지도가 해당 동의 위치로 자동 이동합니다


### 유저가 필터 아이콘을 탭하면 카테고리·동·층 필터 창이 열립니다

- 편의시설 창이 열려 있으면 자동으로 닫힙니다
- 필터 창이 열린 상태에서 필터 아이콘 버튼이 강조 표시됩니다


*(정보 테이블 — Tiptap 원본 보존)*


### 유저가 층 선택기에서 층을 탭하면 해당 층의 지도로 전환됩니다

- 층 목록: B1, 1층~9층 (총 10개, 위에서 9층→B1층 순으로 표시)
- 초기 화면에서는 5층이 보이는 위치로 스크롤되어 있습니다


### 유저가 매장 셀을 탭하면 하단에 매장 정보 카드가 표시됩니다

- 해당 매장이 속한 동이 자동으로 강조됩니다
- 카드에 매장명, 카테고리, 위치 정보가 표시됩니다


### 유저가 하단 매장 카드를 탭하면 해당 매장 상세 화면으로 이동합니다

- 입점 대기 중인 공간(매장 데이터 없음)은 탭해도 상세 화면으로 이동하지 않습니다


### 유저가 매장 카드의 ✕ 버튼을 탭하면 카드가 닫힙니다

- 매장 강조도 해제됩니다


### 유저가 지도 배경을 탭하면 하단 매장 카드가 닫힙니다

- 카드 닫힘과 UI 숨김이 동시에 실행됩니다


### 유저가 편의시설 버튼을 탭하면 편의시설 목록 창이 열립니다

- 필터 창이 열려 있으면 자동으로 닫힙니다
- 편의시설 항목: 화장실, ATM, 엘리베이터, 에스컬레이터, 흡연실, 계단 (6개)


### 유저가 편의시설 항목을 탭하면 선택 또는 해제됩니다

- 편의시설 창이 열려 있어도 지도를 계속 탐색할 수 있습니다


### 검색 또는 필터 결과가 적용된 경우 해당 매장이 지도에서 강조 표시됩니다

- 강조되지 않은 매장은 흐리게 표시됩니다
